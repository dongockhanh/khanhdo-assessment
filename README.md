# khanhdo-assessment

This repository contains multiple independent automation projects. Each project is organized in a separate folder.

## Project Structure

* task1-web-automation/ → UI automation tests using Playwright
* task2-api-automation/ → API automation tests using Playwright

## How to Run

* Run UI Automation task1-web-automation and follow README of this taslk
```
cd task1-web-automation
```

* Run API Automation task2-api-automation and follow README of this taslk
```
cd task2-api-automation
```

* Run API Automation task3-mobile-automation and follow README of this taslk
```
cd task3-mobile-automation
```

* Each project is self-contained. Please refer to the respective README files for detailed setup and execution steps.
